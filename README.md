## ✨ Features

- **📍 Auto-Location Detection**: Automatically detects your city to provide local weather updates instantly.
- **🌡️ Real-Time Metrics**: View current temperature, "feels like" temperature, humidity, and wind speed.
- **📅 7-Day Forecast**: Detailed daily forecast including high and low temperatures with custom weather icons.
- **🌓 Dynamic Theme System**:
    - Smooth 1.2s expansion transition between Dark and Light modes.
    - Custom animated backgrounds that adapt to the time of day and theme.
- **🎨 Immersive Weather Effects**:
    - **Animated Sky**: A canvas-based sky with twinkling stars in dark mode.
    - **Live Particles**: CSS-based rain, snow, and fog layers that activate based on current weather conditions.
    - **3D Micro-interactions**: Interactive metric cards with subtle 3D hover effects.
- **📏 Unit Conversion**: Seamlessly toggle between Celsius (°C) and Fahrenheit (°F).

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Vanilla CSS3 with Modern Features:
    - CSS Variables for easy theming
    - Flexbox & CSS Grid for responsive layouts
    - Glassmorphism effects (`backdrop-filter`)
    - Keyframe animations for weather particles
- **Backend API**: [Open-Meteo](https://open-meteo.com/) (No API key required)
- **Geocoding**: [Nominatim OpenStreetMap](https://nominatim.org/)

## 🚀 Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge).
- An active internet connection for real-time weather data.

### Installation

1. Clone the repository or download the project files.
2. Ensure `index.html`, `style.css`, and `script.js` are in the same directory.

### Usage

1. Open `index.html` in your web browser.
2. Grant location permissions when prompted to automatically load weather for your area.
3. Use the toggle in the top-right corner to switch between Dark and Light modes.
4. Click the temperature units (°C/°F) to switch measurement systems.
